# АИС «Отбор ИИ-проектов» для региона

Веб-платформа для сбора, оценки и отбора проектов с использованием искусственного интеллекта, предназначенных для внедрения в регионе. Система обеспечивает прозрачный процесс: поставщики заполняют самооценку по единым критериям, эксперты региона анализируют и принимают решения, администратор управляет процессом.

---

## 🎯 Основные возможности

### Для поставщика
- Регистрация и подача заявки на ИИ-проект
- Самооценка по 10 критериям с визуальными ползунками
- Подсказки по каждому критерию и методика оценки
- Прикрепление ссылок на файлы (облачные хранилища)
- Отслеживание статуса и балла своей заявки

### Для эксперта региона
- Просмотр всех заявок в едином реестре
- Фильтрация по статусу, сфере, избранному, поиск
- Детализация баллов по каждому критерию
- Действия: одобрить, отклонить, удалить, оставить комментарий
- Сравнение проектов (до 3 одновременно)
- Аналитика: графики динамики и распределения по сферам
- Экспорт в CSV и PDF

### Для администратора
- Управление экспертами (добавление/удаление)
- Просмотр и модерация всех проектов
- Изменение статуса проектов

### Общий функционал
- Авторизация по ролям (поставщик/эксперт/администратор)
- Все данные хранятся в localStorage (легко заменяется на бэкенд)
- Уведомления (имитация мессенджера MAX)
- Адаптивный дизайн (работает на ПК и планшетах)

## 📊 Критерии оценки

| № | Критерий | Вес | Макс. балл |
|---|----------|-----|------------|
| 1 | Социально-экономический эффект | ×2.0 | 5 |
| 2 | Масштабируемость | ×1.6 | 5 |
| 3 | Чёткость KPI | ×1.2 | 3 |
| 4 | Технологическая готовность (TRL) | ×2.0 | 5 |
| 5 | Объяснимость (XAI) | ×1.2 | 3 |
| 6 | Интеграция с ГИС | ×1.2 | 3 |
| 7 | TCO / окупаемость | ×1.4 | 5 |
| 8 | Отечественность (реестр) | ×1.2 | 5 |
| 9 | Кибербезопасность | ×1.0 | 3 |
| 10 | Внешняя поддержка | ×0.5 | 2 |

**Максимальный балл:** 55.8  
**Порог отбора:** ≥28 баллов  
**Автоматическое вето:** TRL <2 ИЛИ Отечественность <2

## 🛠️ Технологии

- HTML5 / CSS3
- JavaScript (ES6)
- Chart.js — визуализация графиков
- jsPDF — генерация PDF-отчётов
- LocalStorage — хранение данных (легко заменяется на бэкенд)

## ⚙️ Установка и запуск

1. Скачайте файл `index.html`
2. Откройте его в любом современном браузере (Chrome, Edge, Firefox, Safari)
3. Начните работу — все данные сохраняются автоматически

**Никаких зависимостей, серверов и интернета не требуется.**

## 🔄 Возможные доработки

При необходимости система легко расширяется:

- Подключение бэкенда (Flask/FastAPI + PostgreSQL)

- Email / Telegram уведомления
- Роль «Наблюдатель» (только просмотр)
